\fbox{\parbox{\dimexpr\linewidth-2\fboxsep-2\fboxrule}{
10-5.
Find the possible values of the quantity represented by the matrix
\begin{equation*}
2\Sigma_1+\Sigma_2+2\Sigma_3
\end{equation*}

Compare this with Problem 4-6.
}}

\bigskip
We have $x_0=0$ and $r=\sqrt{2^2+1^2+2^2}=3$ hence the possible values are
\begin{align*}
x_0+r&=3
\\
x_0-r&=-3
\end{align*}
